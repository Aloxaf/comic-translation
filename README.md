# AI漫画翻译辅助工具
这是一个基于深度学习的漫画翻译辅助工具
包含翻译，图像去字，自动嵌字功能。

目的是帮助非专业汉化人员完成更简单，快速的翻译任务。也可以用于加快专业翻译流程。
ps：机翻有待加强，需要润色。也希望更多的小众漫画能够得到汉化。
## 安装
开发者直接git
注意paddle，pytorch，tensorflow2 需要官网下载，都是cuda10.2的。
